# FinanceApp (Offline-first)

Lightweight Android personal finance tracker built with Kotlin + Compose + Room + Hilt.

## Implemented
- Offline-first local expense + category storage (Room)
- MVVM architecture with repository/use-cases
- Minimal dashboard with total, insights, and charts
- Deterministic offline insights engine
- Hybrid budget planner with optional pluggable LLM API via Retrofit

## Structure
`data/`, `repository/`, `domain/models`, `domain/usecases`, `ui/screens`, `ui/components`, `ui/viewmodels`, `di/`, `utils/`
